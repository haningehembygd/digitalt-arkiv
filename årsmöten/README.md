# Årsmöten

Denna mapp innehåller handlingar från Haninge Hembygdsgilles årsmöten.

Här publiceras kallelser, dagordningar, verksamhetsberättelser, ekonomiska handlingar, motioner, protokoll och övriga dokument som hör till respektive årsmöte. Handlingarna är ordnade årsvis i undermappar och ingår i föreningens digitala arkiv för att ge medlemmar och allmänhet enkel tillgång till föreningens officiella dokumentation.
