# Stadgar

Denna mapp innehåller Haninge Hembygdsgilles gällande och tidigare stadgar.

Stadgarna beskriver föreningens ändamål, organisation och regler för verksamheten. Dokumenten publiceras för att ge medlemmar och allmänhet tillgång till föreningens styrande regelverk. Om flera versioner finns tillgängliga är de ordnade kronologiskt för att bevara föreningens historik.
